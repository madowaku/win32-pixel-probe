# MadoCore 144 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the MadoCore 144 Rust terminal game core described in `madocore144.md`.

**Architecture:** Use a single external-crate-free Rust binary. Keep terminal game logic and Pixel Layer logic in small, deletable sections inside `src/main.rs`, with documentation in `README.md` and `docs/madocore-144.md`.

**Tech Stack:** Rust standard library only, Cargo, Windows terminal output.

---
